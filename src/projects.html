FarmTech

Portable device to monitor the quality of soil.It helps in finding the essential nutrients such as Nitrogen,Potassium required based on crops to be cultivated. It helps in finding nutrients required such as Nitrogen, Potassium, Phosphorus for fields based on crops to be cultivated. I trained a Linear regression model that could predict Nitrogen value in the soil sample. The Learned model has Red, Green and Blue color values as its input and quantity of nitrogen as output. The accuracy of trained model was 72%. I used technologies such as Android Studio for front end, Python for communication between sensors and Android App and Raspberry-pi for Computing.

  • Android Studio
  • Java
  • Python
  • Machine Learning
  • Raspberry-Pi
